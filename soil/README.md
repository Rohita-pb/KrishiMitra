# 🌱 Soil ML Project

## 🚀 Features
- Predict soil health (N, P, K, EC, pH, Moisture)
- Multi-output ML model (XGBoost)
- FPGA-ready LED outputs

## 📂 Project Structure
- main.py → ML model
- Crop_recommendation.csv → dataset

## ⚙️ Setup
pip install pandas numpy scikit-learn xgboost joblib

## ▶️ Run
python main.py
